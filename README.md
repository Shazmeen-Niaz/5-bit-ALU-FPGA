# 5-Bit Registered ALU on FPGA
Synthesizable 5-bit ALU implemented in Verilog HDL on Cyclone V FPGA.

## Operations
- Addition
- Subtraction  
- Multiplication (Structural Shift-and-Add)

## Features
- Registered I/O via D Flip-Flops
- 11-state FSM datapath control
- 100 MHz timing, <1% logic utilization
- Zero compilation errors

## Tools
Quartus Prime, EPWave, ModelSim

## Hardware
FPGA Cyclone V (5CGXFC7C7F23C8)
